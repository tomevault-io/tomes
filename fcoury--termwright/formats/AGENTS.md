# AGENTS

## Commit Instructions

- Use Conventional Commits style for all new commits.
- Preferred format: `<type>(<scope>): <imperative summary>`
- Keep `type` lowercase. Common types in this repo: `feat`, `fix`, `chore`, `debug`.
- Use a short, specific scope when possible (for example: `ghostty`, `tauri`, `ui`).
- Keep the summary concise and action-oriented.

## Examples

- `fix(ghostty): correct HiDPI mouse coordinates in embedded terminal`
- `feat(ui): add project explorer folder interactions`
- `chore(tauri): enable MCP debug capability`
