# AGENTS.md

This workspace is a local operating space for an AI agent helping one user manage AI information overload.

Default principles:

- during initialization, use `self-context/alignment-intake.md` for profile brief and targeted QA notes
- read `self-context/index.md` before judging relevance
- use `config/source-registry.yaml` for source registry entries
- write raw source material to `raw/`
- write retained source summaries to `wiki/sources/`
- promote repeated patterns to `wiki/themes/`
- track recurring entities in `wiki/dossiers/`
- write signals and digests only when useful
- update long-term preferences only from confirmed feedback

Default read order:

1. `AGENTS.md`
2. `self-context/index.md`
3. `wiki/index.md` when existing knowledge may matter
4. task specific README or template files

Do not store credentials, private tokens, cookies, or local secrets in this workspace.
