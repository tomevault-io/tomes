# Claude Code Project Instructions

See [README.md](./README.md) for project overview and the full skill index.

## Conventions

- Each skill lives at `skills/<slug>/SKILL.md` with YAML frontmatter.
- Slugs are kebab-case; skill names in the README match the folder name.
- When adding or editing a skill, update the matching section in README.md (heading, blockquote description, three example prompts).
- Keep descriptions concrete and task-shaped — each example prompt should map to a realistic user request, not a feature list.

## Related packs

Tighter, focused packs live in separate repos: [swe-skills](https://github.com/ckorhonen/swe-skills) (strategic engineering work) and [hone-skills](https://github.com/ckorhonen/hone-skills) (scheduled code hygiene). The shared hub is at [cdd.dev/skills](https://cdd.dev/skills).
