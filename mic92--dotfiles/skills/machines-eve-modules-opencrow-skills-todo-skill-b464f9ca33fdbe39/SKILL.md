# Todo Management

Run `vdirsyncer sync` before reading and after writing.

## Todos (todoman)

```bash
todo list
todo new "<task>"
todo done <id>
```

## Sync (vdirsyncer)

```bash
vdirsyncer sync          # sync all calendars, todos, contacts
```
