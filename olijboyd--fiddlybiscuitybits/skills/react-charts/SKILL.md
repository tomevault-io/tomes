  # React Charts with Recharts

  ## When to Use
  - Creating dashboard chart components
  - Adding data visualisation to React apps
  - Customising Recharts themes and tooltips

  ## Instructions
  1. Always wrap charts in ResponsiveContainer
  2. Use the project's colour palette from Tailwind config
  3. Add meaningful tooltips with custom formatters
  4. Handle empty data states gracefully
  5. Memoize chart data transformations with useMemo
