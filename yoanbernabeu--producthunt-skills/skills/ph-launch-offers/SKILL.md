# Product Hunt Launch Offers

This skill helps you create compelling promotional offers for your Product Hunt launch that drive conversions while staying within community guidelines.

## When to Use This Skill

- Designing your launch promotion
- Deciding between discount types
- Creating urgency and exclusivity
- Structuring lifetime deals
- Maximizing launch conversions

## The Golden Rule

**Offers must NOT be tied to voting.**

✓ Allowed: "Product Hunt community gets 20% off"
✗ Forbidden: "Upvote to get 20% off"

## Types of Launch Offers

### Type 1: Percentage Discount

**Format:** X% off for Y period

**Common Structures:**
- 20% off first year
- 30% off first 3 months
- 50% off first month

**Example:**
```
🎁 Product Hunt Exclusive
30% off your first year
Code: PRODUCTHUNT
Valid for 48 hours
```

**Best For:**
- SaaS products
- Subscription services
- Recurring revenue models

---

### Type 2: Lifetime Deal (LTD)

**Format:** One-time payment for lifetime access

**Common Structures:**
- $99 lifetime (vs $29/month)
- $199 lifetime (vs $49/month)
- Tiered: $49 / $99 / $199

**Example:**
```
🚀 Launch Special: Lifetime Deal
Pay once, use forever

$99 one-time (normally $29/month)
Limited to first 500 users
```

**Best For:**
- Early-stage products
- Building initial user base
- Products with low marginal cost

**Considerations:**
- Attracts deal hunters (may not be long-term users)
- Gives up recurring revenue
- Creates loyal early adopters
- Good for validation/feedback

---

### Type 3: Extended Trial

**Format:** Longer free trial than normal

**Common Structures:**
- 60 days free (vs 14 days)
- 3 months free trial
- "Forever free" during beta

**Example:**
```
Product Hunt Special
60-day free trial (normally 14 days)
No credit card required
```

**Best For:**
- Products needing time to show value
- Complex products with learning curve
- Building habit before payment

---

### Type 4: Feature Unlock

**Format:** Access to premium features for free

**Common Structures:**
- Pro features free for 6 months
- Premium tier at free tier price
- Early access to new features

**Example:**
```
🔓 Unlock Pro Features
Product Hunt users get Pro features
at the Free tier price — forever
```

**Best For:**
- Freemium products
- Products with tiered features
- Upsell-focused models

---

### Type 5: Bundle/Bonus

**Format:** Extra value included

**Common Structures:**
- Templates included free
- Onboarding call included
- Extra credits/usage

**Example:**
```
Product Hunt Bundle
Get [Product] + $500 in templates
+ 1-on-1 onboarding call
All for $99
```

**Best For:**
- Products with add-on value
- Service-based products
- Content/template products

---

### Type 6: Early Adopter Pricing

**Format:** Locked-in lower price forever

**Common Structures:**
- Current price locked forever
- Grandfather early users
- Beta pricing permanently

**Example:**
```
Early Adopter Pricing
Lock in $19/month forever
(Price increases to $39/month soon)
```

**Best For:**
- Products planning to raise prices
- Building loyal early base
- Creating urgency

## Pricing Psychology

### Anchoring
Show the "real" price to make the deal feel valuable.

```
$99/year (normally $199) — 50% off!
```

### Scarcity
Limit availability to create urgency.

```
Limited to first 100 users from Product Hunt
Only 23 spots remaining
```

### Urgency
Time-limit the offer.

```
Expires in 48 hours
Launch week only
Valid through Sunday
```

### Exclusivity
Make PH community feel special.

```
Exclusive for Product Hunt
Not available anywhere else
PH community only
```

## Offer Communication

### In Your PH Listing
- Mention in description
- Feature in gallery image
- Note in maker comment

### In Outreach
```
"For the Product Hunt community, we're offering
[offer details]. This is exclusive to PH and
ends [deadline]."
```

### On Your Website
- Create dedicated PH landing page
- Auto-apply discount if from PH
- Show offer prominently

## Compliance Reminders

### Allowed ✓
- "PH community gets X off"
- "Launch special for PH users"
- "Exclusive offer for Product Hunt"
- "First 100 from PH get..."
- Time-limited offers
- Quantity-limited offers

### Forbidden ✗
- "Upvote to unlock discount"
- "Vote for 50% off"
- "Upvoters get early access"
- Any reward tied to voting

## Offer Planning Checklist

### Before Launch
- [ ] Offer type decided
- [ ] Pricing calculated
- [ ] Discount code created
- [ ] Landing page ready
- [ ] Limits set (time/quantity)
- [ ] Tracking in place

### In Launch Materials
- [ ] Mentioned in description
- [ ] Gallery image created
- [ ] Maker comment includes offer
- [ ] Email templates include offer
- [ ] Social posts include offer

### Technical Setup
- [ ] Discount code works
- [ ] Correct percentage applied
- [ ] Expiration date set
- [ ] Usage limits configured
- [ ] Analytics tracking

## Measuring Offer Success

### Track These Metrics

| Metric | What It Tells You |
|--------|-------------------|
| Code usage count | How many redeemed |
| Conversion rate | Visitors → Customers |
| Revenue per launch | Total launch revenue |
| CAC from PH | Cost to acquire PH customers |

### Post-Launch Analysis
- Did offer drive conversions?
- Was discount too generous?
- Did scarcity work?
- Would you use same offer again?
