# Product Hunt Launch Strategy

This skill helps you create a comprehensive launch strategy for Product Hunt, from defining objectives to creating a detailed preparation timeline.

## When to Use This Skill

- Starting to plan a Product Hunt launch
- Defining launch objectives and success metrics
- Creating a preparation timeline (4-6 weeks recommended)
- Deciding between visibility, acquisition, or social proof goals
- Planning resource allocation for the launch

## Launch Objectives Framework

### Primary Objective Types

1. **Visibility & Brand Awareness**
   - Goal: Maximum exposure to tech community
   - Success metric: Views, impressions, social shares
   - Best timing: Weekdays (Tuesday-Thursday)
   - Required effort: High

2. **Customer Acquisition**
   - Goal: Drive signups, trials, or purchases
   - Success metric: Conversion rate, new users
   - Best timing: Weekdays for B2B, any day for B2C
   - Required effort: Very High

3. **Social Proof (Badge)**
   - Goal: Win "Product of the Day" badge
   - Success metric: Final ranking position
   - Best timing: Weekends for easier competition
   - Required effort: Medium

4. **Qualified Feedback**
   - Goal: Get expert feedback on MVP
   - Success metric: Comment quality, actionable insights
   - Best timing: Any day
   - Required effort: Medium

## 4-6 Week Preparation Timeline

### Week 1-2: Foundation
- [ ] Define primary and secondary objectives
- [ ] Audit current Product Hunt profile
- [ ] Start engaging with PH community (comments, upvotes)
- [ ] Research competitors and similar launches
- [ ] Create supporter list (friends, customers, community)

### Week 3: Content Preparation
- [ ] Write tagline (multiple versions for testing)
- [ ] Write product description
- [ ] Create thumbnail/GIF
- [ ] Prepare gallery images (4+ images)
- [ ] Script demo video
- [ ] Draft maker comment

### Week 4: Outreach Setup
- [ ] Prepare email templates (pre-launch, launch day, follow-up)
- [ ] Draft social media posts
- [ ] Identify communities to share with
- [ ] Consider hunter outreach (optional)
- [ ] Set up analytics tracking

### Week 5: Final Preparation
- [ ] Schedule launch date (specific day chosen)
- [ ] Finalize all assets
- [ ] Test all links and CTAs
- [ ] Prepare launch day discount/offer
- [ ] Create response templates for comments
- [ ] Brief team on launch day roles

### Week 6: Pre-Launch
- [ ] Final review of all materials
- [ ] Schedule posts and emails
- [ ] Notify close supporters of exact launch time
- [ ] Prepare launch day monitoring setup
- [ ] Get rest before launch day

## Strategic Considerations

### Product Readiness Checklist
- [ ] Product is live and accessible
- [ ] Core features work reliably
- [ ] Onboarding flow is smooth
- [ ] Support channels are ready
- [ ] Pricing/plans are finalized

### Team Preparation
- [ ] Assign launch day roles
- [ ] Set up communication channel (Slack, Discord)
- [ ] Create shift schedule for 24-hour coverage
- [ ] Prepare escalation procedures

## Output Format

When using this skill, provide:
1. **Objective Statement**: Clear primary goal with measurable success criteria
2. **Timeline**: Customized week-by-week preparation plan
3. **Resource Assessment**: Team availability and required assets
4. **Risk Factors**: Potential challenges and mitigation strategies
5. **Success Metrics**: Specific KPIs to track
