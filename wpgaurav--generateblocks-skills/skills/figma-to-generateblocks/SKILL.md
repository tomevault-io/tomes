# Figma to GenerateBlocks

Use this skill when converting Figma designs to GenerateBlocks V2 format.

## Trigger Conditions

Activate when user:
- Shares a Figma screenshot or design image
- Provides a Figma URL or exported CSS
- Asks to "convert this Figma design"
- Wants to "implement this design in WordPress"
- References "Figma to GenerateBlocks" or "Figma to GB"

## Limitations

- Cannot directly access Figma files via API (user must provide screenshot, CSS, or description)
- Color extraction from screenshots is approximate
- Font matching depends on web font availability
- Hover states must be inferred (Figma shows static designs)
- Animations/interactions need manual specification

## Required Context

For best results, ask user for:
1. Screenshot or image of the design
2. Any specific colors, fonts, or brand guidelines
3. Whether cards/elements should be clickable
4. Mobile design (if available) or responsive preferences
