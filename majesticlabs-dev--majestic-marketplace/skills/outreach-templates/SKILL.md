# Outreach Templates

Copy-paste templates for organic traffic generation through outreach.

## Guest Post Pitch

```
Subject: Guest post idea: [Specific topic they haven't covered]

Hi [Name],

I've been reading [Blog] for [timeframe] - loved your piece on [specific article].

I noticed you haven't covered [gap topic] yet. I'd love to write about:

"[Proposed title]"

Quick outline:
- [Point 1 with unique angle]
- [Point 2 with data/example]
- [Point 3 with actionable takeaway]

Here's my recent work: [1-2 links]

Would this fit [Blog]?

[Your name]
```

**Follow-up (5 days):**
```
Hi [Name], bumping this in case it got buried. Happy to adjust the angle if [alternative topic] works better. [Your name]
```

## Podcast/Interview Request

```
Subject: [Show name] guest idea: [Your unique angle]

Hi [Host],

Just finished [specific episode] - [genuine specific comment].

I'm [one-line credibility]. I could share:

1. [Unique story/insight]
2. [Contrarian take]
3. [Tactical how-to]

Recent appearances: [1-2 links]

Worth a 15-min chat to see if there's a fit?

[Your name]
```

## Link Building Outreach

```
Subject: Resource for your [article topic] post

Hi [Name],

Found your article on [topic] while researching [related topic].

I created [resource type] that might complement your section on [specific section]:

[One-line description + link]

Either way, great piece - bookmarked it.

[Your name]
```

## Cold Email Sequence (B2B)

**Email 1: Problem-focused**
```
Subject: [Specific problem] at [Company]?

Hi [Name],

[Companies like theirs] often struggle with [specific problem].

Last month I helped [similar company] [specific result with number].

Worth a quick chat to see if relevant?

[Your name]
```

**Email 2: Case study (3 days)**
```
Subject: Re: [Original subject]

Quick follow-up with context:

[Company name] had [problem]. After [solution], they saw [result].

Full breakdown: [link or offer to share]

Relevant to [their company]?

[Your name]
```

**Email 3: Direct ask (5 days)**
```
Subject: Re: [Original subject]

Last try - would [specific small ask] be useful?

If not a fit, no worries. Just let me know either way.

[Your name]
```

## LinkedIn Connection Sequence

**Connection request:**
```
Hi [Name] - saw your post on [topic]. [Genuine observation]. Would love to connect.
```

**Message 1 (after accept, 2 days):**
```
Thanks for connecting! Quick question - what's your biggest challenge with [their focus area] right now?
```

**Message 2 (after response):**
```
Interesting. I've seen [similar companies] solve that by [approach].

I put together [resource] on this - happy to share if useful?
```

**Message 3 (soft pitch):**
```
Based on what you shared, [your solution] might help with [specific problem].

Want me to show you how [similar company] used it? Takes 15 min.
```
