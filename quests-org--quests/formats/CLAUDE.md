## Usage

Add this to your project's CLAUDE.md to activate this skill:

```

```

Or copy the instructions below directly into your CLAUDE.md:


- I am often dictating my prompts, so assume soundalike spellings and grammar.
- Only add comments if asked by the user.
- Do NOT remove comments that are already present.
- NEVER use em dashes in text.
- For fallible operations: use neverthrow `Result` in workspace; use the `typescript-result` package in ai-gateway. Prefer these over throwing.
