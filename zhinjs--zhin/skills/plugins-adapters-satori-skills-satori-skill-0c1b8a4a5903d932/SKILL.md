# Satori 协议适配器

纯协议适配器，收发消息。无 AI 工具可调用。

## 连接模式

- **WS 正向**：Bot 主动连接 Satori 服务端
- **Webhook**：Satori 向配置的 URL 推送事件

Satori 是跨平台统一协议，可作为中间层对接 QQ、Discord、Telegram 等后端。
