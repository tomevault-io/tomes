# 链接海报卡片

中间件自动检测消息中的链接，生成预览卡片。无 AI 工具可调用。

## 支持平台

- **Bilibili**：视频/专栏 → 封面 + 标题 + UP 主
- **GitHub**：仓库/Issue/PR → Star 数 + 描述
- **抖音**：短视频 → 封面 + 标题
- **小红书**：笔记 → 封面 + 标题

依赖 `@zhin.js/html-renderer` 将 HTML 模板渲染为图片（`pnpm add @zhin.js/html-renderer`）。
