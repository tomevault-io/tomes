> - Prefer skills under `.codex/skills` (or `skills/` if linked).

## Usage

Add this to your project's CLAUDE.md to activate this skill:

```
Read and follow the instructions in .claude/skills/rust-skills/SKILL.md
```

Or copy the instructions below directly into your CLAUDE.md:

﻿# AGENTS

- Prefer skills under `.codex/skills` (or `skills/` if linked).
- For Rust questions, route to the most relevant `rust-*` skill.
- Keep answers concise and include runnable code when applicable.
- Suggest validation commands: `cargo check`, `cargo test`, `cargo clippy`.
