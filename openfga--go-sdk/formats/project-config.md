## Architecture

### Dual-Layer Design

1. **Low-Level Layer** (`APIClient`, `APIExecutor`)
   - HTTP request marshaling, retry logic, error handling
   - Direct API responses (raw structs)
   - Use when you need fine-grained control

2. **High-Level Layer** (`OpenFgaClient` in `client/`)
   - Batch request splitting (50 tuples per batch max)
   - Parallel request execution (10 max parallel requests)
   - Credential injection and token refresh
   - Telemetry instrumentation
   - Use for standard operations (Check, BatchCheck, Expand, etc.)

### Package Purposes

- **Root package** (`openfga`): Core types, configuration, APIClient
- **`client/`**: High-level SDK wrapper, method implementations
