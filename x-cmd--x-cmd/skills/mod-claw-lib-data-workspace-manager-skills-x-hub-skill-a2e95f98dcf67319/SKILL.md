# Claw x-hub Skill

## Login

`x hub login` (first time only; skip if already logged in). Commands will prompt for login if needed — just follow the prompt.

## Common Commands

- `x hub --help`: Discover features
- `x hub file upload <path>`: Upload files
- `x hub file share`: Get share links
- Add `--help` to any subcommand when unsure, e.g., `x hub file --help`
