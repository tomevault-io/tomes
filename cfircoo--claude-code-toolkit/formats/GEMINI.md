## Obsidian CLI (active — v1.12.7)
On WSL2, `obsidian` is a wrapper to the Windows Obsidian.com CLI binary.
Prefer CLI commands over raw file reads:

| Task | CLI command | Fallback |
|------|------------|---------|
| Search vault | `obsidian search query=<text> format=json` | Grep files |
| Today's daily note | `obsidian daily` / `obsidian daily:read` | Read file |
| Incomplete tasks | `obsidian tasks todo format=json` | Grep `- [ ]` |
| Find orphans | `obsidian orphans format=json` | Manual link scan |
| Get backlinks | `obsidian backlinks file=<name> format=json` | Grep `[[name]]` |
| Open a note | `obsidian open path=<path>` | N/A |
