# 提交流程 SOP

1. 先使用 `bash` 调用 `git status` 确认当前有哪些文件发生了改动。
2. 你的 commit message 必须使用 Emoji 开头，例如：`🚀 feat: 增加新功能` 或 `🐛 fix: 修复 Bug`。
3. 严禁使用 `git commit -am "update"` 这种敷衍的提交。
