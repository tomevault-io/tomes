# Purpose

Keep documentation accurate, navigable, and aligned with repository reality.

# Workflow

1. Load and apply [docs/ai/CONVENTIONS.md](../../../docs/ai/CONVENTIONS.md), sections 9 and 10.
2. Read affected docs and identify stale references.
3. Update indices and cross-links without duplicating canonical sources.
4. Validate link integrity and naming consistency.
5. Record assumptions and unresolved gaps.

# Inputs

- Docs files in scope.
- Repository structure and scripts as evidence.

# Outputs

- Updated docs files.
- Integrity check result.

# Failure and Escalation

Escalate when source truth is ambiguous or contradictory across canonical docs.
