## Usage

Add this to your project's CLAUDE.md to activate this skill:

```
Read and follow the instructions in .claude/skills/homelab/SKILL.md
```

Or copy the instructions below directly into your CLAUDE.md:

# CLAUDE.md

@AGENTS.md

Notes:
- Repo conventions and known gotchas live in `AGENTS.md` (imported above) — shared
  with any other agent tool.
- Stack-agnostic working standards in the user's global `~/.claude/CLAUDE.md` also apply.
- Project skills are in `.claude/skills/` (a symlink to `.agents/skills/` so they're
  shared cross-tool) — e.g. `/add-app` to scaffold a new application.
