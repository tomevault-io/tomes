# Guidewire Sdk Patterns

## Overview

REST API Client for external integrations, Jutro Digital SDK for React frontends consuming Cloud APIs, Gosu patterns for server-side business logic (entity queries, validation rules, workflows). Use GW REST API Client library for type-safe API calls.
